h1. rvm.el

h2. Overview

h2. Installation

just put rvm.el in your load-path and put the following in your .emacs
<pre>
(require 'rvm)
(rvm-use-default) ;; use rvm's default ruby as default
</pre>

h2. Usage

h3. Switch Ruby Version

you can use the function @rvm-use@ to use another ruby version within emacs. The function @rvm-use-default@ switches to the rvm default version.
